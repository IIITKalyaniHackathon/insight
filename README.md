city-dashboard
==============

Generic Web Dashboard for Urban Based Visualization Projects

This proyect includes these visualizations over maps:

* Heatmaps.
* Polygonal tessellation.
* Graphs.
* Voronoi.
* Polylines.

Check our [tutorials](docs/docs/index.md)!
